# Install

1. Keep the `mv_layer_tools` folder intact.
2. Zip the `mv_layer_tools` folder if you want to install it through Blender's add-on installer.
3. In Blender 4.x, open `Edit > Preferences > Add-ons > Install...`.
4. Choose the zip file, or place the folder in your Blender add-ons directory and refresh.
5. Enable `MV Layer Tools`.
6. Open the 3D View sidebar and look for the `MVLT` tab.

Verified in this workspace:

- Blender 3.6.23 background registration and regression flow
- Import, layer registration, duplication, reordering, grouping, delete
- Timeline key insertion and fade / shake / zoom operators
