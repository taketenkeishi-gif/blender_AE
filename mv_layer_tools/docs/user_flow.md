# User Flow

1. Install and enable the `mv_layer_tools` add-on in Blender 4.x.
2. Open `View3D > Sidebar > MVLT`.
3. In `Scene Tools`, run `Initialize 2D Scene`.
4. In `Import`, use `Import Images` to bring in multiple PNG / JPG assets.
5. In `Layers`, choose the active layer from the list and manage visibility, locking, grouping, duplication, or ordering.
6. In `Timeline`, set `frame_start / frame_end`, jump to boundaries, and add keys.
7. In `Effects`, apply fade, shake, zoom, or direct opacity updates.
